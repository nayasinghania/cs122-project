# Weather Analysis App (CS 122 Summer 2026 Project)

## Authors
- Naya Singhania
- Jaden Navidad

## Description
Our CS 122 project will be a historical weather analysis TKinter desktop application. 
This application will query historical data for a specific time period and location using a freely available weather API.
Upon pulling the specified data from ths weather API, the data will be stored in a CSV file that can then be used for various analysis applications, such as a time series plot or calculations of average, maximum, and minimum temperature. 
The main goal of this application is to provide users insights into historical weather for where they live. 
With this information, they can better understand what future weather may look like in their area. Depending on feasability, this application may show the real-time weather for a selected location as well. 


## Outline
- We will use TKinter to create our user interface
- We will use CSV files to store data pulled from a weather API
- We will use numpy and scipy to analyze temperature trends over a specific period of time
- We will represent this data with a time series plot
